# Improve Strategy Selection And Signal Filtering

## 背景

用户反馈两个问题：

- 策略树点击后缺少明显选中状态。
- 当前策略是趋势回调类时，信号页仍可能展示旧的趋势突破类历史信号，导致检查项数量看起来不跟当前策略一致。

## 需求

- 策略树中当前查看策略和当前使用策略必须有明显标记。
- 信号页默认只展示当前策略生成的信号。
- 可选查看全部历史信号，避免隐藏旧记录。
- 当前策略没有信号时提示用户点击“按当前策略刷新信号”。

## 验收

- 趋势回调类策略下，信号页默认展示该策略的 5 项做多和 5 项做空检查项。
- 如果看全部历史，仍可查看旧策略信号。
- Python 语法检查通过。
