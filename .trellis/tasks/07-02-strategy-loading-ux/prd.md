# Refine Strategy And Loading UX

## 背景

用户希望策略选择不是两个连续 selectbox，而是在一个选择框里直接确定一级大类和二级策略。同时，信号刷新时页面变灰缺少明确 loading 提示。

## 需求

- 策略页顶部选择器改成单个组合选择框。
- 选项文本包含一级大类和二级策略名称。
- 选择一个选项后即可确定完整策略，不再要求先选大类再选子策略。
- 信号刷新按钮执行时显示 loading 提示。
- 不改变策略编辑表单内部的“策略分类”参数编辑能力。

## 验收

- 策略页只有一个用于选择当前查看策略的选择框。
- 信号刷新期间有清晰的 spinner 文案。
- Python 语法检查通过。
