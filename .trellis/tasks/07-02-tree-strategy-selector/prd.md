# Use Tree Strategy Selector

## 背景

用户明确要求策略选择采用树状二级分类，类似“省份 / 城市”区域选择。之前的两个下拉框和平铺组合选项都不符合预期。

## 需求

- 策略页顶部使用树状二级分类。
- 一级节点是策略大类。
- 二级节点是该大类下的具体策略。
- 点击二级策略后展示详情和编辑表单。
- 不再使用平铺的“策略大类 / 策略名”selectbox 作为主选择器。

## 验收

- UI 中能看到大类分组和该分组下的策略。
- 用户可以直接在树状分类里点击具体策略。
- 当前选中策略和当前使用策略有明确提示。
- Python 语法检查通过。
