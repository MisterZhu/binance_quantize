# Make Loading States Visible

## 背景

用户反馈看不到 loading。当前实现使用 `st.spinner`，但部分操作执行很快或随后立即 `st.rerun()`，导致 loading 只是页面短暂变灰，肉眼不明显。

## 需求

- 使用更明显的状态块展示接口操作过程。
- 对关键接口操作保留操作完成结果，避免 rerun 后消失。
- 优先覆盖手动刷新信号、API 认证检测、合约持仓读取等用户可感知接口操作。

## 验收

- 点击关键接口按钮后页面显示明确“进行中/完成/失败”状态。
- 手动刷新信号完成后不立即 rerun 导致提示消失。
- Python 语法检查通过。
